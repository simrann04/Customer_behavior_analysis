# Customer_behavior_analysis
## Overview
This project focuses on performing end-to-end Data Analytics on a dataset using Python, SQL, and Power BI. The workflow includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, dashboard creation, and report generation to extract meaningful business insights.

The project demonstrates practical skills in data analysis, visualization, and reporting using industry-standard tools.

## Dataset
The dataset used in this project contains structured data related to customer/business operations for analytical purposes.

### Dataset Features
- Multiple columns with categorical and numerical data
- Real-world business/customer information
- Suitable for analysis, visualization, and reporting

## Project Workflow / Steps

### 1. Data Loading
- Imported dataset into Python environment
- Checked dataset structure and column information

### 2. Exploratory Data Analysis (EDA)
- Analyzed trends and patterns
- Generated statistical summaries
- Created visualizations for better understanding

### 3. Data Cleaning
- Handled missing values
- Removed duplicates
- Corrected inconsistent data formats
- Processed data for analysis

### 4. SQL Analysis
- Imported cleaned data into database
- Performed SQL queries using:
  - SELECT
  - WHERE
  - GROUP BY
  - ORDER BY
  - JOIN
  - Aggregate Functions

### 5. Power BI Dashboard
- Connected database/dataset to Power BI
- Built interactive dashboards and visual reports
- Added charts, KPIs, slicers, and filters

### 6. Report & Presentation
- Prepared analytical report
- Created project presentation using Gamma
